# ToMoKo

ToMoKo は、ハロー！プロジェクト周辺のイベント、セットリスト、楽曲、メンバー、会場、リリース情報を検索・閲覧するための Web アプリです。

イベントやセットリストを条件で探したり、楽曲の披露回数、メンバーごとの参加情報、会場や地域ごとの傾向、リリース情報などを確認できます。ブラウザ上で公開データを読み込み、検索や集計を行います。

このリポジトリには、ToMoKo の公開ソースコードと公開データを置いています。

## 主な機能

- イベント、ステージ、セットリストの検索
- 楽曲、メンバー、グループ、会場、リリースの詳細表示
- 楽曲ランキングやメンバー検索
- 日付、地域、出演者、曲名などを使った絞り込み
- 公開データを使ったブラウザ内での検索・集計

不具合報告、データ修正依頼、新機能の追加提案、既存機能や表示の改善提案を Issue で受け付けています。少しでも気になることや、協力できそうなことがあれば気軽に投稿してください。

## Issue について

- 同じ内容の Issue がすでにないか確認してください。
- 不具合の場合は、発生したページや操作、見えている内容があると調べやすくなります。
- データ修正の場合は、対象のイベント、日付、曲名、メンバー名などがあると助かります。
- 新機能や改善提案の場合は、何をできるようにしたいか、どの画面や検索で使いたいかを書いてください。
- 個人情報、ログイン情報、認証情報は投稿しないでください。

## 対応範囲

このリポジトリでは、公開サイト上で確認できる機能、表示、データ、公開ソースコードに関する報告や提案を受け付けます。

## 技術構成

- React / TypeScript / Vite
- Tailwind CSS
- DuckDB WASM
- Parquet 形式の公開データ

## 開発

```bash
npm install
npm run dev
```

品質確認:

```bash
npm run typecheck
npm run build
```

Pull Request を送る場合は、対象の Issue や変更理由を説明してください。大きな機能追加は、先に Issue で相談してください。

## 注意

すべての Issue に個別回答や対応を保証するものではありません。
